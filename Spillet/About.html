About!



Preussens Labyrint is a browser game created by Team 23
during a group project in the Informatikk 1 class at NTNU
Trondheim. With little to no experience in game design and
programming, we have spent the last two months coming up
with an idea, creating a plan and finishing the game. We
used javascript for the game, and html and CSS for the
website. We hope you enjoy our game!
